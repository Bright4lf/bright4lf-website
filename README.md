# BRIGHT4LF — Official Website

**Health for Everyone** | Smart Medicine Safety Sensor Technology

Built with React + Vite. Deployed on Vercel.

---

## Local Development

```bash
npm install
npm run dev
```
Open http://localhost:5173

## Build for Production
```bash
npm run build
```

## Deploy to Vercel
1. Push this repo to GitHub
2. Go to vercel.com → Import Project → Select this repo
3. Vercel auto-detects Vite — click Deploy
4. Your site is live at https://bright4lf.vercel.app

## Connect Custom Domain (bright4lf.com)
1. Buy domain at Namecheap, GoDaddy, or Google Domains
2. In Vercel → Project → Settings → Domains → Add bright4lf.com
3. Copy the DNS records Vercel gives you
4. Paste them in your domain registrar's DNS settings
5. Wait 10–30 minutes — done!

## Editing the Site
All content is in: `src/App.jsx`
- Change text, colors, prices, contact details directly in that file
- Save → Vercel auto-redeploys in ~30 seconds
